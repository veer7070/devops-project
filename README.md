# DevOps Full-Stack Application

## Overview
A full-stack application built with React and Flask, containerized using Docker and deployed on AWS EC2 with automated CI/CD using GitHub Actions.

## Tech Stack
- React
- Flask
- Docker
- GitHub Actions
- AWS EC2

## Features
- User management
- Dockerized frontend and backend
- Automated CI/CD pipeline
- Cloud deployment on EC2

## Architecture
Code → GitHub → GitHub Actions → Docker → AWS EC2
